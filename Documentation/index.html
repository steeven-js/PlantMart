Installing Node Modules(Not included due to large upload size)

Well, before running and testing your App for both iOS and Android you need to install Node Modules and Pod Files. We will discuss about installing Pod files in the very next section mentioned below. First, let's install Node Modules. To install Node Modules type the following command shown below in your terminal from the root directory of the project. 

npm install or npm install --force

Or for more simplicity watch this video posted on youtube https://www.youtube.com/watch?v=2FDx7bLzkNU

IMPORTANT: Installation of Node Modules may take a few minutes. It depneds on your Internet speed. npm install command will install all the required NPM Packages mentioned in the package.json file located in the root directory of the project.

Installing Pod Files(Not included due to large upload size)

To install Pod Files you have to navigate to the ios folder first then install the Pod Files here. Type the following command shown below in your terminal from the root directory of the project. 

cd ios && pod install && cd ..

Or for more simplicity watch this video posted on youtube https://www.youtube.com/watch?v=MfPSzgb0a_o

IMPORTANT: Installation of Pod Files may take a few minutes. This command will install all the required Pod Files.

Running iOS App

To run the iOS version of the App make sure that you have already started the Metro server from your terminal and then type the following command in a separate tab of your terminal. 

npx react-native run-ios

Running Android App

To run the Android version of the App make sure that you have already started the Metro server from your terminal and then type the following command in a separate tab of your terminal. 

npx react-native run-android

IMPORTANT NOTE: Above mentioned commands are for npm users only. If you use the yarn package manager then the run command might be a little bit different as compared to the npm run command.

Changing App Logo


The App Logo has been used in several screens and components, So to change the App logo you need to change the following screens and components mentioned below.

1. Drawer Navigation Header Logo

To change the Drawer navigation header logo, You need to open the index.js file located in the src/navigators/drawers/HomeDrawer directory at the root of the App directory. Now you can change the default App logo with yours on Line 58 & 59 shown in the screenshot image given below.

2. Splash Screen Logo

To change the Splash screen logo, You need to open the index.js file located in the src/screens/Splash directory at the root of the App directory. Now you can change the default App logo with yours on Line 15 shown in the screenshot image given below.
